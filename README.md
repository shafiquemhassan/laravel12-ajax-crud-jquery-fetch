# Laravel 12 AJAX CRUD (jQuery + Fetch API)

This project is a simple CRUD application built with **Laravel 12** and **Bootstrap 5**, showing how to handle data using two different approaches:

1. **jQuery AJAX**
2. **JavaScript Fetch API**

Both methods interact with a Laravel API controller, making this a helpful starter template for beginners learning Laravel API development or those practicing client-side CRUD operations.
